Progest is a project manager which focuses on personal projects.

## Prerequisites

- Rust
- Node.js
- pnpm

## Folder Structure

`app/` - Contains the main application code. Uses Tauri and SolidJS.

`cli/` - Contains the CLI code. Uses Rust.

## Development

For development, you can run the following commands:
```bash
make [cli|app]-dev
```

## Build

For building, you can run the following commands:
```bash
make [cli|app]-build
```
or
```bash
make all
```
